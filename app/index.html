Chrome Extension
Sync your profile to the CertApp extension for one-click autofill on government portals.
🧩 How it works

1. Install the CertApp Chrome Extension (load from certapp/extension folder in Chrome → Extensions → Load unpacked)
2. Click "Export to Extension" below — downloads a .certapp file
3. Click the CertApp icon in Chrome toolbar → Import that file
4. Visit any govt portal — click the extension icon → Auto Fill Page

📤 Export Profile to Extension
